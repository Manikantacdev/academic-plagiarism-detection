# AcademicGuard: Intelligent Academic Integrity Platform

AcademicGuard is a modern, full-stack academic integrity solution designed for educators and institutions. It combines advanced semantic plagiarism detection with state-of-the-art AI content analysis to ensure the originality of student submissions.

## ğŸš€ Key Features

*   **Multi-User Isolation**: Secure registration and login system. Every educator has their own private dashboard and reports history.
*   **Dual-Engine Detection**:
    *   **Plagiarism Engine**: Uses FAISS and Sentence-Transformers for high-speed semantic search across a shared knowledge base.
    *   **AI Detector**: Integrated RoBERTa-based models to identify text generated by ChatGPT, Gemini, and other LLMs.
*   **Shared Intelligence**: While user reports are private, all submissions contribute to a global knowledge base to catch cross-user plagiarism.
*   **Interactive Reports**: Side-by-side comparison of submissions with matched sources, highlighting exact and paraphrased matches.
*   **Comprehensive Analytics**: Visual trends for average originality, risk distribution, and submission metrics.

## âš™ï¸ Installation & Setup

### 1. Prerequisites
*   Node.js (v18+)
*   Python 3.12

### 2. Backend Setup
```powershell
# Navigate to project root
cd Detector

# Create and activate virtual environment
python -m venv .venv
.\.venv\Scripts\activate

# Install dependencies
pip install -r backend/requirements.txt

# Run the backend
.\.venv\Scripts\python.exe -m uvicorn backend.main:app --host 127.0.0.1 --port 8000 --reload
```

### 3. Frontend Setup
```powershell
# Navigate to frontend directory
cd frontend

# Install dependencies
npm install

# Run the development server
npm run dev
```
